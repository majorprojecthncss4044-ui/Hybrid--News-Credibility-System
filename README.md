# Hybrid News Credibility Scoring System

## Overview
Developed a Hybrid News Credibility Scoring System that evaluates the trustworthiness of online news articles using Machine Learning, Natural Language Processing (NLP), semantic analysis, and fact-checking techniques. The system analyzes article content from either user-provided text or URLs and generates a comprehensive credibility score along with visual explanations.

## Key Features
- News article extraction and preprocessing from URLs.
- Fake news detection using a trained Machine Learning classifier.
- Semantic similarity analysis using BERT-based models.
- Writing quality assessment based on readability and linguistic features.
- Fact verification through external fact-checking APIs.
- Interactive web interface for real-time analysis and visualization.
- Combined scoring mechanism to generate an overall credibility score.

## Responsibilities
- Collected and processed news datasets for model training and evaluation.
- Implemented text preprocessing and feature extraction pipelines.
- Developed modules for fake news classification and credibility scoring.
- Integrated semantic analysis and fact-checking components.
- Designed and developed the Streamlit-based user interface.
- Performed testing, evaluation, and optimization of the system.

## Outcomes
- Successfully classified news articles as reliable or suspicious using hybrid analysis techniques.
- Improved credibility assessment by combining machine learning predictions, semantic similarity, writing quality metrics, and fact verification results.
- Delivered a user-friendly web application capable of providing real-time credibility analysis of news content.- Combined scoring mechanism to generate an overall credibility score.
